──────────────────────────────────────────────────────────────
RECOMMENDED SPECS
──────────────────────────────────────────────────────────────
  • Format:      PNG (with transparency) or JPG
  • Aspect:      Square (1:1) works best — will be cropped to
                 fit the card via CSS object-fit: cover
  • Size:        ~400×400 px minimum, 800×800 ideal
  • Focus:       Head/shoulders portrait framing looks cleanest

──────────────────────────────────────────────────────────────
FALLBACK BEHAVIOR
──────────────────────────────────────────────────────────────
If an image file is missing, the site automatically falls back
to the original emoji icon (🏹 💀 ⚔️ 🔥 🛡️ 🌿 🗡️) via the
onerror handler in index.html. No broken image icons will show.
